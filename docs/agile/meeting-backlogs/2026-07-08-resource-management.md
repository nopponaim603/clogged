# Meeting Note รอบที่ 4 - Resource Management (Clogged Game Draft 2)

**วันที่:** 2026-07-08

## 1. ภาพรวม
ผู้เล่นเลือกเส้นทางการเคลื่อนที่ของยาน พร้อมวางแผนให้ Units ต่างๆ สำรวจและรวบรวมทรัพยากร เพื่อปราบบอสทั้ง 3 ตัวและจบเกม เกมแบ่งออกเป็น 3 Phase: **Phase ยาน**, **Phase กลางวัน**, **Phase กลางคืน**

## 2. Phase ยาน
- รูปแบบ Node เลือกเส้นทาง มีทั้งหมด 3 ชั้น ชั้นละ 5 block
- ผู้เล่นเริ่มที่จุด Start ของแต่ละชั้น เลือก Node ในแต่ละ block แล้วเดินทางไปยัง Node ของ block ถัดไปตามเส้นทาง/ลูกศรที่เชื่อมกัน
- Node แบ่งเป็น 5 แบบ:
  - **Node ธรรมดา** — ทรัพยากรพื้นฐาน, พื้นที่ล่าสัตว์, โบราณสถาน, Encounter (สุ่มเหตุการณ์ ‘?’), ร้านค้า (แลกเปลี่ยน/ซื้อขาย/Recruit ลูกเรือ) ปรากฏแบบสุ่มตามอัตราการเกิด
  - **Node พิเศษ: ล่าสัตว์** — มีพื้นที่ล่าสัตว์แน่นอน ระดับสัตว์เพิ่มขึ้น
  - **Node พิเศษ: โบราณสถาน** — มีโบราณสถานแน่นอน เพิ่มระดับของ Relic ที่จะได้รับ
  - **Node พิเศษ: ทรัพยากรเฟื่องฟู** — สุ่ม 1 ในทรัพยากรพื้นฐานให้ปรากฏเป็นจำนวนมาก
  - **Node บอส** — มีบอสปรากฏในช่วง Phase กลางคืน (หมายเหตุจากเอกสารต้นฉบับ: ยังไม่ได้คิดรายละเอียดของ Phase กลางวันสำหรับ Node นี้)
- Node พิเศษมีทรัพยากรอื่นๆ ปรากฏเหมือน Node ธรรมดา บวกคุณสมบัติเพิ่มเติมตามประเภท
- เมื่อเลือก Node เสร็จ ผู้เล่นเข้าสู่ Phase กลางวัน

## 3. Phase กลางวัน
- เข้าสู่ Phase นี้หลังเลือก Node จาก Phase ยานแล้ว
- เป็น Map แสดงตำแหน่งยานผู้เล่น ทรัพยากรพื้นฐาน พื้นที่ล่าสัตว์ โบราณสถาน Encounter และร้านค้า ที่ปรากฏแบบสุ่ม
- ผู้เล่นวางแผนให้ Units เดินทางไปเก็บทรัพยากรตามพื้นที่ต่างๆ แล้วกดดำเนินการให้ Units ทำตามแผน ก่อนจบ Phase เข้าสู่ Phase กลางคืน
- สามารถ Upgrade และ Crafting ได้ก่อนดำเนินการ
- Units ใช้ **ระบบเวลา** ในการเคลื่อนที่และ interact กับพื้นที่ต่างๆ — แต่ละ Unit มีเวลาที่ใช้ได้ต่อวันเท่ากัน
- ผู้เล่นได้รับทรัพยากรเมื่อ Unit เดินทางไปเก็บและกลับยานสำเร็จภายในเวลาที่กำหนด
- หาก Unit เดินทางกลับยานไม่ทันเวลา ผู้เล่นจะ**เสีย Unit นั้น**

## 4. Phase กลางคืน
- เข้าสู่ Phase นี้หลังจบ Phase กลางวัน
- มุมมอง top-down ให้ผู้เล่นวางแผนป้องกันยานจากศัตรู/บอสที่มาเป็น Wave จาก 4 ทิศทาง (เหนือ ใต้ ตะวันออก ตะวันตก)
- วิธีป้องกันยาน:
  - ป้อมปืนหลัก (ผู้เล่นควบคุม) โจมตีได้ทั้ง 4 ทิศ — รูปแบบการโจมตีเปลี่ยนตาม Unit ที่ประจำการ
  - ป้อมปืนรอง โจมตีทิศทางที่กำหนด (Automation)
  - ป้อมเสริม ที่ผู้เล่นวาง — อยู่ชั่วคราวเฉพาะคืนนั้น (Automation)
  - Units ที่วางไว้นอกยานเพื่อป้องกัน (Automation)
- ก่อนเริ่มต่อสู้ ผู้เล่นเข้าสู่ช่วงวางแผน โดยจะรู้ล่วงหน้า: จำนวนศัตรู, ทิศทางที่จะโจมตี, และสัญลักษณ์พิเศษหากมีศัตรู/บอสพิเศษในทิศทางนั้น
- สามารถ Upgrade และ Crafting ได้ก่อนจบการวางแผน
- ในช่วงวางแผนผู้เล่นสามารถ:
  - วางป้อมชั่วคราวที่สร้างไว้
  - วางอุปกรณ์ป้องกันอื่นๆ (กับระเบิด, ลวดหนาม, สิ่งกีดขวางเส้นทาง)
  - วาง Units ในเส้นทางที่ศัตรูจะเข้าโจมตี
  - สั่งเคลื่อนย้าย Units ระหว่างเส้นทาง (ใช้เวลาก่อนถึง และระหว่างเคลื่อนที่โจมตีไม่ได้)
- Unit/ป้อมชั่วคราววางได้เฉพาะในพื้นที่สี่เหลี่ยมสีฟ้าที่กำหนด
- สัญลักษณ์แจ้งเตือน: ลูกศรสีแดง = ทิศทาง+ขนาดกลุ่มศัตรู (จำนวนลูกศรที่ซ้อนกัน), ดาวสีแดง = ศัตรูพิเศษ/บอสจากทิศนั้น
- หลังป้องกันสำเร็จ ผู้เล่นเข้าสู่ช่วงสรุปผล แล้วดำเนินต่อที่ Phase ยานในรอบถัดไป

## 5. ระบบย่อย

### 5.1 ระบบเวลา
- เวลาเป็นทรัพยากรที่ใช้วางแผนให้ Units ทำสิ่งต่างๆ ในแผนที่
- หาก Unit ใช้เวลารวมเกินกว่าที่กำหนด จะไม่ได้รับทรัพยากรที่เก็บมา และเสีย Unit นั้นไป
- ตัวอย่างการคำนวณ (Unit ละ 12 หน่วยเวลา):
  - **สถานการณ์ A:** Unit A เดินทางไปพื้นที่ทรัพยากร X (2 หน่วย) → เก็บ X 16 หน่วย (8 หน่วยเวลา, อัตรา 2/หน่วยเวลา) → เดินทางกลับยาน (2 หน่วย) = รวม 12 หน่วย → **สำเร็จ** ได้รับ X 16 หน่วย
  - **สถานการณ์ B:** Unit B เดินทางไปพื้นที่ทรัพยากร Y (3 หน่วย) → เก็บ Y 8 หน่วย (8 หน่วยเวลา, อัตรา 1/หน่วยเวลา) → เดินทางกลับยาน (3 หน่วย) = รวม 14 หน่วย → **เกินเวลา** ไม่ได้รับทรัพยากรและเสีย Unit

### 5.2 ระบบ Units
- Units คือทรัพยากรที่ใช้เก็บทรัพยากร, interact/trigger เหตุการณ์ใน Encounter และร้านค้า, และป้องกันยาน
- แต่ละ Unit มีความสามารถและความถนัดต่างกัน:
  - **ความสามารถ** เพิ่มโอกาส/Bonus เช่น โอกาสได้ Relic, โอกาสได้ทรัพยากร 2 เท่า, ได้ทรัพยากร X เพิ่มเป็น %/จำนวนที่กำหนด, เพิ่มพลังโจมตีเมื่อใช้ใน Phase กลางคืน
  - **ความถนัด** ลดเวลาที่ใช้ทำสิ่งต่างๆ เช่น ลดเวลาเก็บทรัพยากร X เป็น %/จำนวนที่กำหนด
- เสริมค่าพลังให้ Units ได้ผ่านอุปกรณ์สวมใส่และ Relic

### 5.3 ระบบ Upgrade
- นำทรัพยากรมาเพิ่มความสามารถ Units ผ่าน: อุปกรณ์สวมใส่ Units, การเก็บสะสม Relic จากโบราณสถาน

### 5.4 ระบบ Crafting
- นำทรัพยากรมาสร้าง: อุปกรณ์สวมใส่ Units, ป้อมปืนชั่วคราว, อุปกรณ์ป้องกันฐาน (กับระเบิด, ลวดหนาม, สิ่งกีดขวางเส้นทาง)

### 5.5 ระบบ Facility
- นำ Units มาประจำการใน Facility ต่างๆ บนยาน เพื่อเพิ่มความสามารถหรือรับ Bonus ระหว่าง Phase กลางวันและกลางคืน
- (เอกสารต้นฉบับตัดจบตรงนี้ — ส่วนนี้ยังไม่สมบูรณ์)

## 6. Next Steps
- คิดรายละเอียด Phase กลางวันของ Node บอส ให้ครบ (ระบุไว้ในเอกสารต้นฉบับว่ายังไม่ได้คิด)
- ขยายรายละเอียดระบบ Facility ให้จบ (เอกสารต้นฉบับตัดจบกลางประโยค)
- พิจารณาว่า node-map แบบ 3 ชั้น + ship phase นี้ จะแทนที่หรือควบรวมกับแนวทาง crew dispatch แบบง่ายที่ prototype ปัจจุบันใช้อยู่หรือไม่
