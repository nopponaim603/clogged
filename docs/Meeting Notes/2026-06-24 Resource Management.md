## 1. แนวทางเกม (Genre แนวทางที่พิจารณา)

### 1.1 Resource Management
**Fun Factor**
- การบริหาร assets ที่มีอยู่อย่างจำกัด ให้ผ่านได้
- Puzzle resource management: ใช้ assets ที่มีในการผ่านด่าน
- การหา currency พิเศษบางอย่างมาใช้ในการผ่านด่าน / อัพเกรดทรัพยากร

### 1.2 Factory Management
- คล้ายกับการสร้างสมการเคมี การนำสิ่งของมารวมกันสร้างของใหม่ จะมีเศษเหลือจากการสร้างด้วย

### 1.4 Idle Game
- ข้อควรระวัง: goal หรือ end game อาจควบคุมได้ยาก

**Fun Factor**

- ความรู้สึกว่าเราไม่ติดขัดอะไรแล้ว เติมเต็มความเติบโต

### 1.5 Clicker
**Fun Factor**
- คลิกไปเรื่อยๆ ก็จะได้ resource มาอัพเกรด
